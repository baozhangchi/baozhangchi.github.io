C# 多线程异步下载文件

Zhangchi Bao
03-16
开发 › C# › 异步 › 多线程
C#, 多线程, 开发, 异步

在C#中，可以使用多线程异步方式下载文件以提高下载速度。以下是一个简单的示例：

using System.Net;
using System.Threading.Tasks;

public static async Task DownloadFileAsync(string url, string filePath, int numThreads = 4)
{
    // 获取文件长度
    HttpWebRequest request = (HttpWebRequest)WebRequest.Create(url);
    request.Method = "HEAD";
    HttpWebResponse response = (HttpWebResponse)await request.GetResponseAsync();
    long contentLength = response.ContentLength;
    response.Close();

    // 计算每个线程应该下载的字节数
    long blockSize = contentLength / numThreads;

    // 创建下载任务列表
    var tasks = new List<Task>();
    for (int i = 0; i < numThreads; i++)
    {
        long startRange = i * blockSize;
        long endRange = (i == numThreads - 1) ? contentLength - 1 : (i + 1) * blockSize - 1;

        tasks.Add(DownloadBlockAsync(url, filePath, startRange, endRange));
    }

    // 等待所有任务完成
    await Task.WhenAll(tasks);
}

private static async Task DownloadBlockAsync(string url, string filePath, long startRange, long endRange)
{
    using (WebClient client = new WebClient())
    {
        // 设置下载范围
        client.Headers.Add("Range", $"bytes={startRange}-{endRange}");

        // 下载文件块并写入文件
        using (var stream = await client.OpenReadTaskAsync(url))
        using (var fileStream = new FileStream(filePath, FileMode.OpenOrCreate, FileAccess.Write, FileShare.Write))
        {
            stream.CopyTo(fileStream);
        }
    }
}

在上述代码中，首先使用HEAD请求获取文件长度。然后，计算每个线程应该下载的字节数，创建一个下载任务列表。对于每个任务，使用Range头指定下载范围，并使用OpenReadTaskAsync方法异步下载文件块。然后将文件块写入文件流中。最后，使用WhenAll方法等待所有任务完成。
