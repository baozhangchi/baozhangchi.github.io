C#监控Windows系统硬件接入断开

Zhangchi Bao
03-16
开发 › C# › Windows
C#, Windows, 开发

可以使用C#中的ManagementEventWatcher类来监控Windows系统硬件接入和断开事件。以下是一个简单的示例代码，该代码可以监控系统中所有硬件接入和断开事件：

using System;
using System.Management;

namespace HardwareMonitoring
{
    class Program
    {
        static void Main(string[] args)
        {
            // Create a new event query to monitor all hardware connect and disconnect events
            WqlEventQuery query = new WqlEventQuery("SELECT * FROM Win32_DeviceChangeEvent");

            // Create a new event watcher and subscribe to events that match the query
            ManagementEventWatcher watcher = new ManagementEventWatcher(query);
            watcher.EventArrived += new EventArrivedEventHandler(DeviceChangeEvent);

            // Start monitoring
            watcher.Start();

            // Wait for user input to exit
            Console.WriteLine("Press any key to exit");
            Console.ReadKey();

            // Stop monitoring
            watcher.Stop();
        }

        static void DeviceChangeEvent(object sender, EventArrivedEventArgs e)
        {
            // Extract event properties
            PropertyData eventType = e.NewEvent.Properties["EventType"];
            PropertyData deviceName = e.NewEvent.Properties["DeviceName"];

            // Print event information
            Console.WriteLine("Event Type: {0}", eventType.Value);
            Console.WriteLine("Device Name: {0}", deviceName.Value);
            Console.WriteLine();
        }
    }
}

在上面的代码中，DeviceChangeEvent方法是一个事件处理程序，它将在硬件接入和断开事件发生时被调用。该方法提取事件的属性，并将其打印到控制台上。您可以根据自己的需求修改此方法，以执行特定的操作。请注意，您需要具有管理员权限才能运行此代码，以便监视系统级别的事件。
